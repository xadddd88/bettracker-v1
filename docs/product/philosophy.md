# Product Philosophy

## Product

BetTracker AI is a Decision Support Platform for sports betting decisions.

It helps users:
- structure betting decisions
- understand probabilities
- track reasoning
- choose Place / Watch / Skip
- connect decisions to bets
- evaluate performance over time

## Future Brand

Future brand: LineHunter AI

Slogan:

> Hunt the edge. Beat the line.

RU:

> Ищи слабые линии. Принимай сильные решения.

## Core Principle

Decision first. Bet optional. Accountability always.

## Product Loop

```
AI Analysis
→ Decision
→ User action: Place / Watch / Skip
→ Optional Bet
→ Settlement
→ Performance Metrics
→ Learning
```

## What We Are Not

BetTracker AI is not:
- a guaranteed prediction engine
- a gambling promise machine
- a black-box tipster
- a bookmaker
- a financial advisor

## AI Honesty Rule

AI Analyst must always include an honesty disclaimer.
The system may identify reasoning, probability, edge, risk, and uncertainty.
The system must not guarantee outcomes.
